# Magic Layout Tool – Purpose

Magic is an open-source VLSI layout editor used for:
- Drawing IC layouts
- Performing Design Rule Check (DRC)
- Preparing layouts for LVS verification
- Supporting technology files like SKY130

Magic is widely used for educational and
research purposes in VLSI backend design.

## Technology Used: 
```
SKY130 PDK
```

## Magic is used in this project for:

✅Creating CMOS circuit layouts.<br>
✅Running Design Rule Check (DRC).<br>
✅Preparing layouts for LVS verification.<br>

This tool forms the foundation of
backend VLSI layout learning.
