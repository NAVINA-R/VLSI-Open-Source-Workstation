# Ubuntu Installation

## OS Version
Ubuntu 20.04 LTS

## Installation Type
- Normal Installation
- Minimal third-party software
- No dual boot

## Post Installation Checks
```bash
lsb_release -a
uname -r
df -h
free -h

```
## Why Ubuntu 20.04?
- Stable LTS release

- Widely supported by open-source VLSI tools

- Compatible with SKY130 PDK and OpenLane
