# System Requirements

## Host Machine
- Operating System: Windows 10 / 11
- RAM: Minimum 8 GB (4 GB allocated to VM)
- Storage: Minimum 100 GB free disk space
- Processor: Intel / AMD with virtualization enabled

## Virtual Machine
- VM Software: VMware Workstation
- Guest OS: Ubuntu 20.04 LTS
- RAM Allocation: 4 GB
- CPU Cores: 2
- Disk Space: 80 GB (Dynamic allocation)

## Reason for Using VM
- Hardware limitations for dual boot
- Easy backup and portability
- Safe experimentation without affecting host OS
