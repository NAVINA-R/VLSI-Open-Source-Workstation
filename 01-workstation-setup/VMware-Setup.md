# VMware Workstation Setup

## Steps
1. Install VMware Workstation on Windows
2. Create a new virtual machine
3. Select Ubuntu 20.04 ISO
4. Configure hardware:
   - RAM: 4096 MB
   - CPU: 2 cores
   - Disk: 80 GB (split into multiple files)

## Important Settings
- Enable virtualization in BIOS
- Enable 3D acceleration (optional)
- Use NAT network mode

## Why VMware?
VMware provides stable performance and good
Linux compatibility for EDA tools.
